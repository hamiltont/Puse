**_Puse_** is a Sequence diagram editor based on [PlantUML](http://plantuml.com/) framework

## Compilation

The following will download dependencies, compile the project, and run the GUI. 
Note: windows users should use `gradlew.bat`

```
$ ./gradlew run
```
